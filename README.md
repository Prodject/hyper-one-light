# hyper-one-light

> [Atom One Light][one-light] theme for [Hyper][hyper]

![](screenshot.png)

## Install

Open your Hyper preferences and add `hyper-one-light` to plugin list:

```js
plugins: [
  'hyper-one-light'
],
```
